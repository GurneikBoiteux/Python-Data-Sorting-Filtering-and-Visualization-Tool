Overview
========
The main purpose of this software is to: load, manage, and visualize data from spreadsheets that contain detailed information about students. It help users to sort and analyze student records.

Dependencies
============

    Python
    Matplotlib
    Numpy
    SciPy

Installation
============

a) Python
----------
1. Verify if python is already installed with the command
    python --version
2. If the command doesn't return the version, download python form (https://www.python.org/downloads/). Follow the installation steps on the website that match your opersating system (like Linux, Windows, or MacOS).

b) Matplotlib
-------------
To install Matplotlib run the command
    python -m pip install matplotlib
Note: python installer will install pip command as well.

c) Numpy
--------
Numpy should be installed through Matplotlib installation.To ensure run the command
    python -m pip install numpy
Note: if numpy isn't installed, pip download and install it with the previous command.

d) SciPy
--------
Run the follow pip command to install SciPy
    python -m pip install SciPy
